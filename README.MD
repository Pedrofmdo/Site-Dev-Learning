Documentação Oficial do projeto do site de vendas CactoDev

- Oque consiste?

* O Site funciona da seguinte forma, será um site de vendas de produtos onlines, para estudo e etc focada ná área da tecnologia.

* Publico voltado para Pessoas que tem interesse em estudos, e aprender.


Estrutura de pastas do projeto: 


seu-site-vendas/
│
├── index.html              (Página principal - arquivo HTML)
├── style.css               (Estilos - arquivo CSS)
├── script.js               (Interatividade - arquivo JavaScript)
│
├── assets/                 (Pasta para todos os recursos)
│   ├── images/             (Imagens e ícones)
│   │   ├── logo.png        (Logo do site)
│   │   ├── favicon.ico     (Ícone da aba do navegador)
│   │   └── produtos/       (Imagens dos produtos)
│   │       ├── curso1.jpg
│   │       ├── curso2.jpg
│   │       └── ...
│   │
│   ├── fonts/              (Fontes personalizadas - se necessário)
│   └── icons/              (Ícones SVG específicos)
│
├── produtos/               (Pasta para páginas de produtos individuais - opcional)
│   ├── curso-fullstack.html
│   └── ...
│
└── README.md               (Documentação do projeto)
